How to use this tool (2 minutes)

📘 Download the full scoring handbook (Word) — detailed guidance on how to read GRADE and AMSTAR-2 and how to score each Bradford Hill criterion.

  1. For each edge you'll see the exposure → outcome, its effect estimate, study quality (GRADE), review quality (AMSTAR-2), number of studies, and the source review.
  2. Score the six Bradford Hill criteria by clicking 0, 1, or 2. The plain-English meaning of each score is printed on the button — just pick the one that fits.
  3. Click “Why / SAM guidance” under any criterion if you want the detailed rule.
  4. Add a note or tick Flag for discussion whenever you're unsure — those become the adjudication list.
  5. The total (0–12) and provisional tier update live. When done, click Export results and send the downloaded file back to the review team.
