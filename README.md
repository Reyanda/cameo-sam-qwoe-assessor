# CAMEO-SAM — QWoE Independent Assessor Scoring

Independent Bradford Hill scoring of causal edges for the CAMEO-SAM evidence synthesis
(severe acute malnutrition, children 6–59 months; PROSPERO CRD420261365391).

## For assessors

1. **Open the scoring tool** (the page served by this repository).
2. Enter your name when prompted, then score the six Bradford Hill criteria (0 / 1 / 2)
   for each causal edge. The plain meaning of every score is on screen.
3. For detailed guidance on reading GRADE and AMSTAR-2 and on scoring each criterion,
   read the handbook: **CAMEO-SAM_Assessor_Handbook.docx**.
4. When finished, click **Export results** and return the downloaded file to the review team.

Each assessor scores independently and does not see anyone else's scores; this allows the
inter-rater agreement (ICC) to be reported, as the protocol requires. Your scores are held
only in your own browser and in the file you export — nothing is uploaded.
